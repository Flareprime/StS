# Spot the Signs

An interactive human trafficking awareness activity developed as a Software Development Capstone project at Columbia Basin College.

## Purpose

Spot the Signs is a web-based educational tool designed for use at community outreach events and awareness booths. Participants use their mobile devices to complete short interactive activities that challenge misconceptions and increase awareness of human trafficking warning signs.

## Features

- Myth vs Fact activity
- Safe or Sus? scenario activity
- Mobile-friendly design
- QR code access
- JSON-driven content
- No login required

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages
